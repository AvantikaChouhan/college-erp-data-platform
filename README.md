# College ERP Analytics Platform

## Project Overview

This project is an end-to-end Data Engineering project that simulates a College ERP system.

The project will generate college data, store it in MySQL, process it using Python and PySpark, and build dashboards for analytics.

## Tech Stack

- Python
- MySQL
- Git
- GitHub
- Azure
- PySpark
- Power BI
